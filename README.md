# HowToUseGithub
###1.Git Add
```
git add .
```
###2.Commit
```
git commit -m "ข้อความที่ต้องการแสดง"
```
###3. ถ้ายังไม่ได้ตั้งค่า git
```
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
```
###4. Git Push
```
git push origin master
```
*จะมีให้ใส่ username & password
